# React App (Next.js)

## Run Locally
- Node 20
- `npm install`
- `npm run dev`

## Build & Start
- `npm run build`
- `npm start`

## Pages
- `/` Home, `/financial-aid`, `/graduate-programs`, `/undergraduate-programs`, `/contact`, `/search`
- Search mount: `#searchstax-root` on `/search`

## Netlify
- Base: `apps/react`
- Build: `npm run build`
- Publish: `.next`
- Plugin: `@netlify/plugin-nextjs`
- Node: 20
